### SCRAPING-FALABELLA

Proyecto de ejemplo de scraping para obtener productos de Falabella mediante Puppeteer y Cheerio.
Toma el primer producto de cada categoría denominada como "oferta".  

Obtiene:
- Nombre
- Precio
- Precio anterior
- Descuento
- Especificaciones
- Links a URLs de imágenes
- Breadcrumbs  

Instalación
- npm install
- node index.js
